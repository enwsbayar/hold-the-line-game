# Hold the Line

![Hold the Line](img.png)

A 2D isometric auto-battle card placement game made with Godot 4.6.

🎮 **[Play Online](https://hold-the-line-game.netlify.app)**

## Gameplay

- Drag character cards onto the battlefield during the prep phase
- Characters automatically fight incoming enemy waves
- Survive as many waves as possible
- Each wave gets harder — enemies scale up and gain bonus stats

## Cards

- **Warrior cards** — place fighters on the battlefield (Bronze → Diamond rarity)
- **Passive cards** — instant effects like healing, stat boosts, or killing an enemy

## Built With

- Godot 4.6
- GDScript
- Pixel art assets
